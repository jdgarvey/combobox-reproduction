# Combobox Reproduction
Minimal reproduction for an issue where using `getByRole('combobox', {name: 'someName'})` does not work.

# Get started

## Locally
```bash
npm install
# In one terminal window
npm start
# In another terminal window
npm test
```

Open http://localhost:3000.
